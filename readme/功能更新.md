# ApolloTab 功能更新记录

## v0.3.9 (2026-06-14)

### 修复
- **空白小节点击定位不准**: 修复 `build_timeline()` 中空小节(无拍/音符)被 `continue` 跳过的问题
  - 原因：原代码对无 beats 的空白小节直接跳过，导致该小节无 timeline 条目，用户点击时定位到相邻有音符的小节，产生较大位置偏差
  - 影响：含休止符/空小节的谱子，点击这些区域时光标跳到错误位置
  - 方案：为每个空白小节生成占位条目(placeholder)，记录正确的 `scroll_y` 和 `time_ms`

### 新增
- **全局唯一小节ID (`global_meas_idx`)**: 时间线索引新增跨系统/跨页的全局唯一小节标识
  - 每处理完一个小节递增一次，确保 ID 在整首曲子中不重复
  - 支持通过 `find_measure_at_time()` 精确按全局ID回溯定位到任意小节的起始位置
- **`find_measure_at_time()` 方法**: 根据时间或滚动位置查找对应小节的详细信息
  - 返回 `{'global_meas_idx', 'meas_idx', 'start_time_ms', 'start_scroll_y'}`
  - 用于 UI 层实现"点击谱面 → 跳转到对应小节播放"的功能

### 优化
- **哨兵点索引继承**: 时间线末尾的哨兵点现在继承最后真实条目的完整小节索引信息（而非硬编码为0）
  - 修复 B点设置在谱子末尾时 `_find_measure_at_time` 返回 meas_idx=0 的错误

### 修改文件
| 文件 | 变更说明 |
|------|---------|
| `player.py` | 新增 find_measure_at_time()；build_timeline() 重构(空白小节占位+全局ID+哨兵修复)；更新头部注释 |

## v0.3.7 (2026-06-13)

### 修复
- **GTPPlayer 缺少 set_theme 方法**: 修复 `'GTPPlayer' object has no attribute 'set_theme'` 错误
  - 原因：v0.2.6 在 TabRenderer 中添加主题系统时，未在门面类 GTPPlayer 中同步暴露接口
  - 影响：用户无法通过 player 对象直接切换渲染主题

### 新增
- **GTPPlayer 主题管理接口**: 新增 3 个委托方法，完整暴露 TabRenderer 的主题功能
  - `set_theme(theme)`: 切换渲染主题（支持字符串/ThemeConfig 实例）
  - `current_theme_name` (property): 获取当前主题名称
  - `get_available_themes()`: 获取所有可用预设主题列表
  
### 设计模式
- **委托模式(Delegate)**: GTPPlayer 将主题设置请求转发给内部 TabRenderer 实例
- **门面模式(Facade)**: 对外提供统一简化的接口，隐藏内部复杂性

### 使用示例
```python
from ApolloTab.player import GTPPlayer

player = GTPPlayer()
player.load("song.gp5")

# 切换到黑白主题（适合打印）
player.set_theme("light")
images = player.render_track(0)  # 使用新主题渲染

# 查看当前主题
print(player.current_theme_name)  # 输出: "light"

# 查看所有可用主题
print(player.get_available_themes())  # 输出: ['light', 'dark']
```

### 修改文件
| 文件 | 变更说明 |
|------|---------|
| `player.py` | 新增 set_theme()/current_theme_name/get_available_themes() 三个方法；更新头部注释 |

## v0.3.6 (2026-06-13)

### 修复
- **调号拍号不显示**: 修复 `_format_key_time_signature()` 访问 `song.key_signature`（不存在）而非 `song.key`（正确字段名），导致方法返回空字符串

### 新增
- **标题区调号拍号显示**: 页面标题右侧新增 `1=X (A/B)` 格式的调号拍号信息
  - 调号自动转换为调性名称（如 C/G/D/E/F#/Bb/Eb 等，支持纯五度循环全映射）
  - 拍号以分数形式展示（分子/分母）
  - 新增 `_format_key_time_signature()` 方法处理转换逻辑

### 修改文件
| 文件 | 变更说明 |
|------|---------|
| `renderer/tab_renderer.py` | `_draw_header()` 标题右侧新增调号拍号；新增 `_format_key_time_signature()` 静态方法 |

## v0.2.6 (2026-06-13)

### 新增
- **渲染主题系统**: 新增 `ThemeConfig` 类，支持黑白(light)和深色(dark)两套内置配色方案
  - `TabRenderer.set_theme("light"|"dark")` 接口支持运行时动态切换主题
  - 支持自定义主题扩展（部分颜色覆盖 + 自动填充默认值）
  - 所有颜色参数统一从 `self.cfg.theme.COLOR_*` 获取

### 修复
- **A4宽高比修正**: 默认渲染尺寸从 `900×1200`(比例1.333) 修正为 `1000×1414`(比例√2=A4标准)
- **颜色访问路径规范化**: 渲染器中所有颜色访问从 `self.cfg.COLOR_*` 统一改为 `self.cfg.theme.COLOR_*`
- **移除调试代码**: 清除 `set_theme()` 中的生产环境 print 语句

### 修改文件
| 文件 | 变更说明 |
|------|---------|
| `utils/constants.py` | 新增 ThemeConfig 类；RenderConfig 默认尺寸改为 A4 比例 |
| `renderer/tab_renderer.py` | 颜色访问改用 theme 对象；新增 set_theme() 接口 |
| `__init__.py`, `utils/__init__.py` | 导出 ThemeConfig |
| `pyproject.toml` | 版本号 0.2.5 → 0.2.6 |
| `README.md` | 新增主题系统文档；版本号更新 |
